# Not Alone — App Prototype

This is a combined React project containing all the Not Alone screens:
- Home feed + Live Map
- Clubs (list + detail, including Music)
- More Flows (Create Club, Chat, Connection Request)
- Profile + Sign Up

This is a **front-end prototype only** — there is no real backend, database, or
payment processing wired in. It's meant to demo the look, feel, and flow of the
app in a browser, and to give a developer a working starting point.

## Run it locally (optional, requires Node.js installed)

```bash
npm install
npm run dev
```

Then open the URL it shows (usually http://localhost:5173).
